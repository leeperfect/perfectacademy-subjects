# 面试技能目录

## 目录结构

- **teaching_skills/**：面试授课技巧
  - 课堂教学方法
  - 面试技巧讲解
  - 互动模拟训练
  - 案例分析方法

- **training_methods/**：面试培训方法
  - 培训课程设计
  - 学员评估方法
  - 个性化培训方案
  - 培训效果评估

- **case_analysis/**：面试案例分析
  - 真题解析案例
  - 学员作答分析
  - 高分答案分析
  - 常见错误案例

- **experience/**：面试教学经验
  - 教学心得分享
  - 培训经验总结
  - 学员反馈分析
  - 教学改进措施

## 内容规范

- **文件命名**：`[技能类型]_[具体内容]_[版本号]_[日期].md`
  - 示例：`teaching_skills_面试结构化技巧_v1.0_20260406.md`

- **内容结构**：
  1. 技能概述
  2. 具体方法
  3. 应用场景
  4. 案例说明
  5. 注意事项

## 版本管理

- 使用语义化版本号：`v[主版本].[次版本].[修订号]`
- 每次更新需在CHANGELOG.md中记录变更内容
- 定期回顾和更新技能内容，保持时效性
